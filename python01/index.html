【Python】小白学Python遇到的问题整理汇总

总结下自己在学习Python过程中遇到的一些错误与解决办法

1.关于路径

1、绝对路径

　　绝对路径有三种使用方法：

　　反斜杠 ‘\’：由于反斜杠 ‘\’ 要用作转义符， 所以如果要使用反斜杠表示路径，则必须使用双反斜杠。

　　原始字符串 r’’：可以使用原始字符串+单反斜杠‘\’的方式表示路径

2.关于with open路径

这里Pythong编程从入门到实践 书里写的是直接当前目录文件名，但我的老师报错，找不到文件，参考后发现改成绝对路径可以读取

参考:Python自学：[Errno 2] No such file or directory报错？

with open(r’d:/Pythong编程从入门到实践/chapter10/pi_digits.txt’) as file_object:

​ contents = file_object.read()

​ print(contents)

2.print取消换行

print (i,end=’’)

3.ModuleNotFoundError: No module named ‘bs4’

from bs4 import BeautifulSoup

ModuleNotFoundError: No module named ‘bs4’

缺模块，安装即可

解决方法：cmd输入pip install bs4

缺啥安啥

ModuleNotFoundError: No module named ‘xlrd’

pip install xlrd

ModuleNotFoundError: No module named ‘xlwt’

pip install xlwt
